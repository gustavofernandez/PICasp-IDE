PICasp-IDE

IDE y programador de escritorio para microcontroladores PIC de Microchip. Editor, compilador XC8, programador USB y monitor serie, todo en uno.

Capturas de pantalla

Características

🖥️

IDE integrado

Editor Monaco (VS Code), compilación con MPLAB XC8, monitor serie y gestor de librerías vía GitHub.

⚡

Programador USB

Soporte para USB-ICSP via Arduino Uno/Nano, Bootloader USB-Serial para PICasp Boards y Config Words (Fuses).

📦

HAL estilo Arduino

API picasp.h con objetos Serial, Wire, SPI, EEPROM y DAC. Estructura setup() / loop().
